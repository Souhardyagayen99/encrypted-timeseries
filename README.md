# encrypted-timeseries

## Folder Structure

### Backend:
  - Emitter.js
  - Listener.js

### Frontend:
  - Components
    - DataDisplay.js
  - index.js

## Running the backend Services

1. To run the Emitter service, navigate to the `backend` folder:

```
cd backend
npm start
```

2. To run the Listener service, navigate to the backend folder:

```
cd backend
nodemon Listener.js
```


## Running the frontend Services

```
cd frontend
npm start
```
